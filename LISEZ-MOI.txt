Q1 : Quand on demande le niveau de jeu de l'ordinateur, on veut pouvoir
répondre « distrait », « naïf », « rapide », etc.  On remarque que le premier
caractère correspond bien à la lettre à taper pour le niveau considéré.
Est-ce possible dans votre programme ?  Dans la négative, comment le modifier
pour que ce soit possible ?
nous devons modifier Demander_niveau de cette maniére :
*Utiliser une chaîne de caractères pour lire l'entrée de l'utilisateur au lieu d'un caractère unique.
*Comparer cette chaîne avec les noms complets des niveaux pour déterminer le choix de l'utilisateur
procedure Demander_Niveau is
   Niveau_Char : String(1..10);  -- Permet de lire jusqu'à 10 caractères
begin
   Put_Line("Niveau de l'ordinateur (naïf, distrait, rapide ou expert) ?");
   Get_Line(Niveau_Char);
   if Niveau_Char = "naïf" then
      Niveau := Naif;
   elsif Niveau_Char = "distrait" then
      Niveau := Distrait;
   elsif Niveau_Char = "rapide" then
      Niveau := Rapide;
   elsif Niveau_Char = "expert" then
      Niveau := Expert;
   else
      Niveau := Expert;  -- Niveau par défaut si l'entrée est incorrecte
   end if;
end Demander_Niveau;
